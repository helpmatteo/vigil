# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in vigil, please report it responsibly.

**Do not open a public issue.**

Instead, email **matteohe.tech@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 72 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.
